Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML5

HTML5 (HyperText Markup Language, version 5) es la quinta revision importante del lenguaje basico de la Worl Wide Web, HTML.

Usa la extencion.html

  • CSS

Hoja de estilo en cascada o CSS (siglas en ingles de cascading style sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML.

Usa la extencion css

  • JS

JavaScript (abreviado comunmente "JS" es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente (client-side), simlemente como parte de un navegador web permitiendo mejoras en la interfaz de usuario y paginas web dinamicas.

Usa la extencion .js
